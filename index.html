HTML Table

Tag Name Description
table Table An organized box with content
th Table Header Naming the rows
td Table Data What goes inside the box
tr Table Row Organizing into rows with content

CSS Table

Tag Name Description
@font-face Font Face Goes on your first line of CSS to give links to the fonts you are using
.selector Class Selector Applies a rule to a specified class in the HTML file
color Foreground Color Defines the forground color for an element
text-align Text Align Where the text is aligned to. Can be center, justified, right, absolute, etc.
